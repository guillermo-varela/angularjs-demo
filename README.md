# angularjs-demo
Demonstration on how to develop a web project using AngularJS 1.5 with some recommended practices.

- Just run from the root folder:
 - npm install
 - npm install -g protractor
 - bower install
 - gulp
- Then open "http://localhost:8080" in your web browser.
